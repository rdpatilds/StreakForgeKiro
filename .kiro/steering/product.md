# Product Overview

## Product Purpose
StreakForge is a personal habit tracking application that helps users build and maintain positive habits through streak tracking and progress visualization. It provides motivation through visual progress indicators and streak counters, making habit formation engaging and rewarding.

## Target Users
- **Primary**: Individuals looking to build better personal habits (exercise, reading, meditation, etc.)
- **Secondary**: Friend groups who want to share and encourage each other's habit-building journeys
- **Needs**: Simple, offline-first tracking without complex authentication or hosting requirements

## Key Features
- **Habit Creation & Management**: Add, edit, and organize personal habits
- **Daily Check-ins**: Simple interface to mark habits as completed each day
- **Streak Tracking**: Visual streak counters that motivate continued consistency
- **Progress Visualization**: Charts and graphs showing habit completion over time
- **Friend Sharing**: Optional ability to share progress with friends locally
- **Offline-First**: Works without internet connection using local SQLite database

## Business Objectives
- Help users successfully build lasting habits through consistent tracking
- Provide motivation through visual progress and streak gamification
- Maintain simplicity and ease of use without complex setup requirements
- Enable social motivation through friend sharing capabilities

## User Journey
1. **Setup**: User opens StreakForge and creates their first habits
2. **Daily Use**: User checks in daily, marking completed habits
3. **Progress Review**: User views streaks, charts, and progress over time
4. **Social Sharing**: User optionally shares achievements with friends
5. **Long-term Engagement**: User maintains habits through visual motivation and streak preservation

## Success Criteria
- Users consistently track habits for 30+ days
- Average streak length increases over time
- Users report improved habit consistency
- Simple setup and daily use (under 2 minutes per day)
- High user retention and engagement with progress features
